# CAN Firmware Downloader

Библиотека для приёма и программирования прошивки через CAN-интерфейс. Предназначена для загрузки firmware во флеш-память микроконтроллера.

## Описание

Данный модуль обеспечивает приём данных прошивки, передаваемых по CAN-протоколу, их буферизацию и последующую запись во флеш-память по секторам. Поддерживается потоковый режим приёма с автоматической синхронизацией блоков данных.

## Структура потока данных

Данные прошивки передаются в следующем формате:

### Заголовок (18 байт)
| Смещение | Размер | Описание |
|----------|--------|----------|
| 0 | 2 байта | Ключ (0x08AA) |
| 2 | 16 байт | Зарезервировано (8 слов по 2 байта) |

### Блоки данных

Каждый блок имеет следующую структуру:

| Поле | Размер | Описание |
|------|--------|----------|
| Точка входа (Entry Point) | 4 байта | Адрес начала выполнения после загрузки (только в первом блоке) |
| Размер блока | 2 байта | Размер блока в 16-битных словах |
| Адрес назначения | 4 байта | Адрес во флеш-памяти для загрузки блока |
| Данные | переменный | Непосредственно данные прошивки (размер = размер блока * 2 байта) |

### Завершение загрузки
Блок с размером = 0 сигнализирует об окончании передачи данных.
